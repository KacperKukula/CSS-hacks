Pozycjonowanie elementu na środku

Pozycjonowanie elementu absolutnego

Pozycjonowanie elementu w centrum ekranu przyspożyło mi wiele nerwów. Przeszukując internet natrafiłem na wiele konceptów. Najlepszy okazał się sposób z pozycjonowaniem elementu abolutnego. 


    .element {
        postionion: absolute;
        top: 0; right: 0; bottom: 0; left: 0;
        margin: 0;
    }

Centrowanie za pomocą flex-box

Można też rozwiązać ten problem poprzez funkcję flex-box, która oferuje pozycjonowanie horyzontalne (horizontal) jak i wertykalne (vertical)

W tym celu w kontenerze (container) implementujemy element (element) i dodajemy kontenerowi atrybury: 


    .container {
        display: flex;
        justify-content: center;
        align-items: center;
    }
    .element {
        display: block;
    }

Centrowanie - inne

Możemy się zdecydować na trzecie rozwiązanie. Jest to centrowanie "sztywne", tzn. przez atrybuty takie jak padding, margin oraz atrybuty pozycjonowania absolutnego (np. top, right). 


    .static1 {
        margin: 50% 50%;
    }
    .static2 {
        padding: 50% 50%;
    }
    .absolute {
        top: 50%;
        left: 50%;
    }

Pamiętajmy też, że atrybuty padding oraz margin różnią się od siebie jednym szczegółem. Margin określa odległość od innego elementu. Padding określa odległość węzła tekstowego (tekstu) od rodzica (w którym jest napisany).
